sabre/katana
==========

sabre/katana is a CalDAV, CardDAV and WebDAV ready-to-use server on top of
[SabreDAV](http://sabre.io/).

Installation
------------

Make sure you have [composer][1] installed, and then run:

    composer require sabre/katana
